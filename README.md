# AI Learning Notes

AI와 관련된 학습 내용을 정리하는 개인 노트/블로그 레포지토리입니다.

## 구조

```
ai-learning/
├── learnings/              # Claude Code와의 대화에서 배운 내용
│   └── YYYY-MM-DD-*.md    # 날짜별 학습 노트
├── topics/                 # 주제별 정리
│   ├── claude-code/
│   ├── llm/
│   └── prompting/
└── resources/              # 참고 자료
    ├── links.md
    └── tools.md
```

## 사용 방법

Claude Code의 `learning-summary` skill을 사용하여 자동으로 학습 내용을 문서화합니다.

```
"지금까지 대화 정리해줘"
"배운 내용 문서화"
```

## 목적

- AI 관련 지식 축적
- 대화에서 배운 내용 보존
- 나중에 참고할 수 있는 구조화된 노트
- 개인 AI 학습 블로그
